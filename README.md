# Kanban MVP

Single-board Kanban MVP built with Next.js in `frontend`.

## Run

```bash
cd frontend
npm install
npm run dev
```

## Test

```bash
cd frontend
npm run lint
npm run test
npm run test:e2e
```

## Scope

- One board with five renameable columns
- Cards with title and details only
- Drag/drop between columns
- Add and delete cards
- No persistence, no auth, no extra features
